# security-camp2026

セキュリティキャンプ 2026 の記録を、見返しやすいようにカテゴリごとへ整理したリポジトリです。

## 構成

- `日記/`: 日ごとに整理した Markdown の日記
- `メモ/`: 講義中の走り書きや発表準備用のラフメモ
- `提出物/`: 修了課題や感想文など、提出用にまとめた文章

## 方針

元メモと整理済みの文章を分けておくことで、あとから見返すときに「まず完成版を見る」「必要ならラフメモに戻る」という流れを取りやすくしています。トップレベルはできるだけ薄く保ち、カテゴリ別ディレクトリに寄せる運用を想定しています。

各ディレクトリの見どころは、それぞれの `README.md` から追えるようにしています。
